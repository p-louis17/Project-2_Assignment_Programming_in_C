# Dynamic Student Record Engine

A menu-driven C program that manages student academic records using **structures**, **dynamic memory allocation**, **pointers**, and **function pointers**.

## Features

* Add student records
* Display all students
* Sort students by grade
* Calculate the average grade
* Find the top-performing student
* Classify students into letter grades (A–F)

## Concepts Used

* Structures
* Dynamic Memory (`malloc` / `free`)
* Pointers
* Function Pointers
* Arrays
* Bubble Sort
* String Handling

## How to Compile

```bash
gcc student_database.c -o student_database
```

## How to Run

```bash
./student_database
```

## Sample Menu

```
1. Add Student
2. Display Students
3. Sort Students
4. Calculate Average
5. Find Top Student
6. Classify Students
7. Exit
```

## Learning Objectives

This project demonstrates how to build a dynamic student management system while practicing:

* Dynamic memory allocation
* Pointer traversal
* Function pointers
* Structures
* Menu-driven programming in C
