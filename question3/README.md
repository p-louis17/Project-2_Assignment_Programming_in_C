# Callback-Based Device Monitoring Simulator

A C program that simulates monitoring multiple devices using **structures**, **unions**, **dynamic memory allocation**, **pointers**, and **callback functions**.

## Features

* Simulate 10 random device readings
* Monitor temperature sensors
* Monitor pressure sensors
* Monitor battery voltage
* Display device status (Normal, Warning, or Danger)
* Process devices using callback functions

## Concepts Used

* Structures
* Unions
* Dynamic Memory (`malloc` / `free`)
* Pointers
* Function Pointers (Callbacks)
* Arrays of Structures
* Random Number Generation

## How to Compile

```bash
gcc device_monitor.c -o device_monitor
```

## How to Run

```bash
./device_monitor
```

## Learning Objectives

This project demonstrates how to build a callback-based device monitoring system while practicing dynamic memory allocation, pointer traversal, unions, structures, and function pointers in C.
