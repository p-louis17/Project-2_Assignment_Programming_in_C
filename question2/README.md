# Adaptive Text Intelligence Tool

A menu-driven C program that analyzes text using **structures**, **dynamic memory allocation**, **pointers**, and **function pointers**.

## Features

* Count unique words
* Find the longest word
* Find the most frequent word
* Calculate the average word length
* Display the memory address of the dynamically allocated word array

## Concepts Used

* Structures
* Dynamic Memory (`malloc` / `free`)
* Pointers
* Function Pointers
* String Manipulation (`strlen`, `strcmp`, `strcpy`, `strtok`)
* Pointer Traversal

## How to Compile

```bash
gcc text_analyzer.c -o text_analyzer
```

## How to Run

```bash
./text_analyzer
```

## Sample Menu

```
1. Word Count
2. Longest Word
3. Most Frequent Word
4. Average Word Length
5. Exit
```

## Learning Objectives

This project demonstrates how to process and analyze text dynamically while practicing dynamic memory allocation, pointer traversal, string manipulation, structures, and function pointers in C.
